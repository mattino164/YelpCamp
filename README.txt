This website is a full-stack web application that demonstrates my knowledge of modern web development. It includes front-end technologies like HTML, CSS, and JavaScript, as well as back-end frameworks such as Node.js and Express. The application utilizes MongoDB as the database, with Mongoose for data modeling. Additionally, it incorporates authentication with Passport.js, session management, and security enhancements using Helmet and MongoSanitize. The project follows RESTful principles and includes features like user authentication, campground listings, reviews, and flash messaging for better user experience.
